# 🤖 Press WhatsApp Bot

A simple WhatsApp bot built with Node.js using the `whatsapp-web.js` library.

## 📋 Requirements

- Node.js (v14 or higher)
- npm or yarn
- WhatsApp account

## 🚀 Installation

1. **Clone the repository:**
```bash
git clone https://github.com/Reynoldspress50-ctrl/press-whatssap-bot.git
cd press-whatssap-bot
```

2. **Install dependencies:**
```bash
npm install
```

3. **Start the bot:**
```bash
npm start
```

## 🔐 How to Use

1. Run `npm start`
2. A QR code will appear in your terminal
3. Open WhatsApp on your phone and go to **Settings → Linked Devices → Link a Device**
4. Scan the QR code with your phone
5. The bot is now connected!

## 📝 Available Commands

- `hi` - Say hello to the bot
- `help` - View available commands
- `time` - Get the current time

## 🛠️ Development

To run the bot in development mode with auto-reload:
```bash
npm run dev
```
